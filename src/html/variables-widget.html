Variables: A Place to Store and Change a Value

Think of a variable like a labeled box that can hold a value (a number, some words, or true/false). You can look inside the box, change what's inside, or move the value to another box.

Try This Exercise:
Current value of the variable score:
0
You can change the number and press Set score or press Add 1 to increase it.
This is what the JavaScript code looks like when we use a variable:
let score = 0;
        score = 5; // changed to 5
        score = score + 1; // add 1
Mini challenge: Set the score to 10, then press Add 1 three times. What is the final number? Try writing it down and then check the box.
Short explanation

- let score = 0; makes the variable called score and puts the number 0 inside it.
- To change it, we assign a new value: score = 5;
- To increase it by one we do: score = score + 1; (this reads: put the old score plus one back into score).

Variables give programs memory so they can remember things while running.
